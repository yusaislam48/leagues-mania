# Leagues Mania

Project Live Link: [Visit Leagues Mania](https://leaguesmania.netlify.app).

## Core Featuures

    *Showing the league name in the home page.
    *You can explore the all league by click them.
    *In the details page you can find more details about this league
    *You can know about founded, country, sports type and gender of this league
    *Details page image and logo will be change.

## Usage Api in this project

Go [The Sports DB Website](https://www.thesportsdb.com/api.php) to view the apis.
